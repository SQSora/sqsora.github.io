创建并初始化Hexo项目

如果你没有Hexo,点击可查看新建Hexo项目

创建Github SSH密钥

Github已不再支持HTTP PUSH,点击可查看生成SSH秘钥并上传至GitHub

在 GitHub 上添加部署 Action 私钥

  • 在GItHub中创建一个代码仓库(可选择Private)

  • 在 GitHub 仓库里，进入 Settings -> Deploy keys -> Add deploy key 页面。在 Title随便输入一个名称,例如HEXO_DEPLOY_KEY,将公钥粘贴到 Key 文本框中,并且选择Allow write access。

  • 如果出现Key is already in use则需要多创建一个密钥
  • 接下来将github_hexo.pub的内容重新安装上面的步骤上传

在 GitHub 上添加部署 Action 私钥：仓库访问 Settings -> Deploy keys -> New respository secret，名字部分填写：HEXO_DEPLOY_KEY​，注意大小写，这个后面的 GitHub Actions Workflow 要用到，一定不能写错。
